# PRJ001 | Productivity Clock

A simple looping countdown clock with customizable period. The application is built via the followings:
- Python 3.8
- Qt for Python (Pyside2)
- QML

## Features
- Countdown clock
- Period customization
- Automatic looping
- Start/Stop button
- Notification sound support
